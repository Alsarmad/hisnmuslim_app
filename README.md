# قم بتشغيل التطبيق في وضع التطوير

<br><br>

**متطلبات التشغيل**

- nodejs

- git

- npm

<br>

```bash
git clone https://github.com/rn0x/hisnmuslim_app
cd hisnmuslim_app
npm install
npm run dev
```

<br>

 **build**

<br>

```bash
#Windows
yarn run packwin
#Linux
yarn run packlinux
```
